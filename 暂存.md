# Capped Nuclear Norm MSLM 完整求解过程

  

---

  

## 一、原始问题与松弛

  

### 1.1 原始非凸非线性问题

  

纱线自动计数的核心假设是：对齐后的子图投影曲线组成的数据矩阵可以分解为低秩部分（共性纱线结构）与稀疏部分（局部异常）。由此得到优化问题：

  

```

min        rank(L) + lambda * ||S||_0

L, S, t

  

s.t.       P(D ∘ t) = L + S                               (1)

```

  

其中：

- D ∈ R^{d×k} 是 k 个子图 ROI 的列向量拼成的数据矩阵，d = m × n

- t = {t_1, ..., t_k} 是每个子图的几何参数（垂直平移量）

- D ∘ t 表示对每个子图施加几何变换后再提取 ROI

- P ∈ R^{m×d} 是投影矩阵（将 d 维列向量投影为 m 维行和向量）

- L ∈ R^{m×k} 是低秩分量，S ∈ R^{m×k} 是稀疏分量

  

问题 (1) 中 rank(L) 和 ||S||_0 都是非凸、不连续的，且 D ∘ t 关于 t 非线性，直接求解不可行。

  

### 1.2 原始 MSLM 的凸松弛（张楠珊论文 式 4.6）

  

将 rank(L) 松弛为核范数 ||L||_*，将 ||S||_0 松弛为 L_{2,1} 范数：

  

```

min        ||L||_*  +  lambda * ||S||_{2,1}

L, S, t

  

s.t.       P(D ∘ t) = L + S                               (2)

```

  

其中：

  

```

||L||_*     = sum_{i=1}^{min(m,k)} sigma_i(L)             核范数（奇异值之和）

  

                  m

||S||_{2,1} = sum     ||S_{i,:}||_2                        L_{2,1} 范数（逐行 L2 范数之和）

                 i=1

```

  

这是一个凸优化问题（在固定 t 时），可以用 ADMM 求解。但核范数对所有奇异值等权惩罚，会过度压缩纱线主体结构对应的大奇异值。

  

### 1.3 本文的非凸紧松弛：Capped Nuclear Norm

  

我们将核范数替换为截断核范数（Capped Nuclear Norm），得到更紧的秩近似：

  

```

                min(m,k)

min        sum         min(sigma_i(L), epsilon)  +  lambda * ||S||_{2,1}

L, S, t      i=1

  

s.t.       P(D ∘ t) = L + S                               (3)

```

  

其中 epsilon > 0 是截断阈值。

  

### 1.4 Capped Nuclear Norm 为什么优于核范数

  

```

截断核范数 vs 核范数 vs 秩函数 对奇异值 sigma 的惩罚：

  

        sigma       rank(惩罚)    ||·||_*(惩罚)    min(sigma, eps)(惩罚)

        ——————      ——————————    ——————————————    ————————————————————

        0           0             0                 0

        0.5*eps     1             0.5*eps           0.5*eps      ← 小 SV：三者接近

        eps         1             eps               eps          ← 分界点

        2*eps       1             2*eps             eps          ← 大 SV：截断停止惩罚

        10*eps      1             10*eps            eps          ← 大 SV：核范数继续增长

```

  

优势总结：

  

(a) **比核范数更紧**：对大奇异值（纱线主结构）的惩罚封顶为 epsilon，不会过度压缩信号。核范数的惩罚随 sigma 线性增长，迫使模型缩小有用信号。

  

(b) **比 rank 更易求解**：rank 是 0-1 阶跃函数，不连续、不可微。Capped Norm 连续且分段线性，可通过 DC（差凸）分解转化为一系列凸子问题。

  

(c) **稀疏项保持 L_{2,1} 范数不变**：L_{2,1} 范数对于纱线图像的物理意义（逐行稀疏 → 只有少数行存在异常）已经足够，无需截断。若对 S 也加截断，会引入额外的 DC 分解层，增加求解复杂度，而收益有限。

## 二、增广拉格朗日函数

  

对问题 (3) 构造增广拉格朗日函数：

  

```

                         min(m,k)

L_mu(L, S, t, Y) =  sum         min(sigma_i(L), epsilon)  +  lambda * ||S||_{2,1}

                        i=1

  

                     + <Y, P(D ∘ t) - L - S>

  

                     + (mu / 2) * ||P(D ∘ t) - L - S||_F^2            (4)

```

  

其中 Y ∈ R^{m×k} 是拉格朗日乘子，mu > 0 是罚参数。

## 三、交替方向乘子法（ADMM）求解

  

采用两层循环结构：外层更新几何参数 t，内层在固定 t 下交替更新 L、S、Y。

  

### 3.1 外层循环：更新几何参数 t

  

与原始 MSLM 完全相同。在当前 t 处：

  

(1) 对每个子图 i，用当前 t_i 提取 ROI 并归一化为 D 的第 i 列

  

(2) 计算竖向 Sobel 梯度，构建雅克比矩阵 J_i

  

(3) 对 P * J_i 做 QR 分解：P * J_i = Q_i * R_i

  

### 3.2 内层循环：固定 t，交替更新 L、S

  

记 PD = P * D 为投影后的数据矩阵。以下每一步的 d 为内层迭代编号。

  

---

  

#### 3.2.1 更新 L（第 d+1 步）——DC 分解 + 加权奇异值软阈值

  

固定 S_d、Y_d、dt_d，L 子问题为：

  

```

                    min(m,k)

L_{d+1} = arg min   sum       min(sigma_i(L), epsilon)

             L        i=1

  

           + (mu_d / 2) * ||L - Z_L||_F^2                             (5)

```

  

其中：

  

```

Z_L = PD + dt_d + (1 / mu_d) * Y_d - S_d                             (6)

```

  

**DC 分解**：将截断核范数拆分为"凸 - 凸"的形式：

  

```

min(sigma, epsilon) = sigma - max(sigma - epsilon, 0)

```

  

因此：

  

```

min(m,k)                                      min(m,k)

 sum     min(sigma_i, epsilon) = ||L||_*  -    sum     max(sigma_i - epsilon, 0)

  i=1                                           i=1

                                  ————————      ————————————————————————————————

                                   凸函数               凸函数 h(L)

```

  

**线性化凹部分**：在当前迭代点 L_d 处，对 -h(L) 做线性化。

  

设 L_d 的紧奇异值分解为 L_d = U_d * diag(sigma_d) * V_d^T，则 h(L) 在 L_d 处的次梯度为：

  

```

partial h(L_d) = U_d * diag(g_d) * V_d^T                              (7)

  

        | 1,    若 sigma_{d,i} > epsilon    （大奇异值）

g_{d,i}=|

        | 0,    若 sigma_{d,i} < epsilon    （小奇异值）

```

  

代入 (5)，DC 线性化后 L 子问题化为：

  

```

L_{d+1} = arg min  ||L||_*  -  <G_d, L>  +  (mu_d / 2) * ||L - Z_L||_F^2     (8)

             L

```

  

其中 G_d = U_d * diag(g_d) * V_d^T。

  

**化简**：展开二次项并合并与 L 相关的线性项：

  

```

(mu_d / 2) * ||L - Z_L||_F^2  -  <G_d, L>

  

= (mu_d / 2) * ||L||_F^2  -  mu_d * <Z_L, L>  -  <G_d, L>  +  const

  

= (mu_d / 2) * ||L  -  (Z_L + G_d / mu_d)||_F^2  +  const

```

  

因此 (8) 等价于：

  

```

L_{d+1} = arg min  ||L||_*  +  (mu_d / 2) * ||L - Z_L'||_F^2                 (9)

             L

  

其中  Z_L' = Z_L + G_d / mu_d

```

  

这是**标准的核范数近端问题**，解为奇异值软阈值算子（SVT）：

  

```

对 Z_L' 做 SVD：  Z_L' = U * diag(sigma) * V^T

  

L_{d+1} = U * diag( max(sigma_i - 1/mu_d, 0) ) * V^T                (10)

```

  

**等价的加权 SVT 视角**：上述过程等价于直接对 Z_L 做加权 SVT（无需显式计算 Z_L'）：

  

```

对 Z_L 做 SVD：  Z_L = U * diag(sigma) * V^T

  

定义权重：  w_i = 1 - g_{d,i}

  

            | 1,    若 sigma_{d,i}(L_d) < epsilon    （小 SV：正常压缩）

    w_i   = |

            | 0,    若 sigma_{d,i}(L_d) >= epsilon   （大 SV：不惩罚）

  

L_{d+1} = U * diag( max(sigma_i - w_i / mu_d, 0) ) * V^T             (11)

```

  

式 (11) 即为代码中实现的版本。

  

---

  

#### 3.2.2 更新 S（第 d+1 步）——L_{2,1} 近端算子

  

固定 L_{d+1}、Y_d、dt_d，S 子问题为：

  

```

S_{d+1} = arg min  lambda * ||S||_{2,1}  +  (mu_d / 2) * ||S - Z_S||_F^2     (12)

             S

  

其中  Z_S = PD + dt_d + (1 / mu_d) * Y_d - L_{d+1}                  (13)

```

  

这与原始 MSLM 完全相同，解为逐行收缩：

  

```

                        lambda

S_{d+1, i,:} = max(1 - ———————————————, 0) * Z_{S, i,:}              (14)

                      mu_d * ||Z_{S, i,:}||_2

  

对 i = 1, 2, ..., m

```

  

**此步与原始 MSLM 无任何区别。**

  

---

  

#### 3.2.3 更新几何增量 dt（第 d+1 步）

  

固定 L_{d+1}、S_{d+1}、Y_d，dt 子问题为：

  

```

                                  k

dt_{d+1} = arg min  ||P * sum    J_i * dt_i * e_i * e_i^T  +  T||_F^2        (15)

              dt              i=1

  

其中  T = (1 / mu_d) * Y_d + P(D ∘ t) - L_{d+1} - S_{d+1}

```

  

利用 P * J_i = Q_i * R_i 的 QR 分解，得到闭式解：

  

```

dt_{d+1, i} = Q_i^T * T_{:,i}                                        (16)

  

dt_matrix_{:,i} = Q_i * dt_{d+1, i}                                  (17)

```

  

**此步与原始 MSLM 无任何区别。**

  

---

  

#### 3.2.4 更新对偶变量 Y 和罚参数 mu

  

```

Y_{d+1} = Y_d + mu_d * (PD + dt_{d+1} - L_{d+1} - S_{d+1})          (18)

  

mu_{d+1} = rho * mu_d                                                (19)

```

  

其中 rho > 1 是罚参数增长因子（本文取 rho = 1.25）。

#### 3.2.5 内层收敛判断

  

```

         ||PD + dt_{d+1} - L_{d+1} - S_{d+1}||_F

crit = ——————————————————————————————————————————— < tol_inner        (20)

                      ||PD||_F

```

  

### 3.3 外层更新 t 并判断收敛

  

内层收敛后，将几何增量累加到 t：

  

```

dt_i = R_i^{-1} * dt_i                     （从投影空间映射回参数空间）

  

t_i(3) = t_i(3) + dt_i                     （累加垂直平移量）           (21)

  

对 i = 1, 2, ..., k

```

  

外层收敛判断使用 Capped 核范数的目标函数值：

  

```

                 min(m,k)                       m

obj = sum               min(sigma_i(L), eps) + lambda * sum ||S_{i,:}||_2     (22)

                  i=1                            i=1

  

当 |obj_{prev} - obj_{cur}| < tol_outer 或达到最大迭代次数时停止。

```

## 四、与原始 MSLM 求解过程的对比

  

### 4.1 改动一览

  

| 求解步骤 | 原始 MSLM | 本文 Capped MSLM | 变化 |

|---|---|---|---|

| L 更新 | 标准 SVT：统一阈值 1/mu | 加权 SVT：大 SV 阈值=0，小 SV 阈值=1/mu | **核心改动** |

| S 更新 | L_{2,1} 行收缩 | 完全相同 | 无变化 |

| dt 更新 | QR 投影 | 完全相同 | 无变化 |

| Y 更新 | 对偶梯度上升 | 完全相同 | 无变化 |

| 外层 t 更新 | 累加平移量 | 完全相同 | 无变化 |

| 收敛判断 | ||L||_* + lambda * ||S||_{2,1} | Σ min(sigma_i, eps) + lambda * ||S||_{2,1} | 目标函数变化 |

  

### 4.2 相比原始 MSLM 的优势

  

**(a) 更紧的秩近似**

  

核范数是 rank 的凸包络（最紧的凸下界），但它仍然距 rank 较远。Capped Nuclear Norm 是一个非凸但更紧的近似。

  

直观理解：rank 函数只关心"有多少个非零奇异值"（0-1 计数），不关心它们的大小。核范数把大奇异值的"大小"也纳入惩罚，导致信号被压缩。Capped Norm 通过封顶消除了这种副作用。

  

**(b) 保护纱线主体结构**

  

在 L 更新的 SVT 步骤中，大奇异值（对应纱线周期图案）的阈值为 0，直接通过不压缩。小奇异值（对应噪声）的阈值为 1/mu，正常压缩。效果：投影曲线的峰谷对比度不会被削弱。

  

**(c) 自适应截断阈值**

  

通过能量占比法自动计算 epsilon（见第五章），不同图像使用不同的阈值，适配性强于固定参数。

  

**(d) 计算代价几乎不增**

  

每次 L 更新只需要额外计算一次 svd(L) 来获取当前奇异值（用于确定权重 w），以及一次逐元素比较。SVD 的矩阵规模为 m × k（如 200 × 20），计算时间可忽略。

## 五、截断阈值 epsilon 的自适应计算

  

### 5.1 能量占比法

  

在首次外层迭代中，对投影数据矩阵 PD 做 SVD：

  

```

sigma_1 >= sigma_2 >= ... >= sigma_r = svd(PD),    r = min(m, k)

```

  

计算累积能量占比：

  

```

                    sum_{j=1}^{i} sigma_j^2

energy_ratio(i) = ——————————————————————————                          (23)

                    sum_{j=1}^{r} sigma_j^2

```

  

找到满足 energy_ratio(k_0) >= eta 的最小 k_0（eta 为能量保留比例，如 0.7），令：

  

```

epsilon = 0.99 * sigma_{k_0}                                         (24)

```

  

### 5.2 物理意义

  

累积能量占 eta 以上的前 k_0 个奇异值被视为"纱线主体结构"，其对应的权重 w_i = 0（不惩罚）。剩余奇异值被视为"噪声"，权重 w_i = 1（正常压缩）。

## 六、收敛性分析

  

### 6.1 内层 ADMM：固定权重下的凸问题

  

当 DC 权重 w 在某次内层迭代中固定后，L 子问题 (9) 是凸的（加权核范数 + 二次罚项），S 子问题 (12) 是凸的（L_{2,1} + 二次罚项）。标准两块 ADMM 的收敛性成立。

  

### 6.2 DC 重加权：非凸的全局收敛

  

权重 w 随 L 的奇异值变化，使得完整的 L 子问题 (5) 是非凸的。但 DC 框架保证：

  

(1) 每步 DC 线性化得到的凸子问题的最优解，使原始非凸目标函数值不增

  

(2) Capped Nuclear Norm 有下界（>= 0）

  

(3) 因此目标函数值单调递减且有下界，必收敛

  

### 6.3 外层循环：含几何变换的非凸问题

  

D ∘ t 关于 t 非线性，使得整体问题非凸。求解策略与原始 MSLM 相同：

  

(1) 对 P(D ∘ t) 在当前 t 处做一阶 Taylor 展开（局部线性化）

  

(2) 近端更新保证每步目标函数充分下降

  

(3) 在 KL（Kurdyka-Lojasiewicz）性质假设下，迭代序列收敛到非凸问题的一个临界点

  

严谨表述：

  

> 在带近端项的交替更新框架下，算法生成的目标函数值序列单调下降并收敛。在 KL 性质假设下，迭代序列收敛到问题 (3) 的一个临界点。

## 七、完整算法伪代码

  

```

算法：Capped Nuclear Norm MSLM

  

输入：二值化子图 {img_1, ..., img_k}，ROI 尺寸 m × n，

      初始几何参数 t，截断阈值 epsilon（或由能量占比法自动计算）

  

输出：低秩分量 L，稀疏分量 S，几何参数 t

  

1.  预计算投影矩阵 P 和 Sobel 梯度 Iy

  

2.  repeat（外层循环）

  

3.      按当前 t 提取 ROI，构建 D、J、Q、R

  

4.      PD = P * D

  

5.      若首次迭代且 epsilon 未指定：由式 (23)-(24) 自适应计算 epsilon

  

6.      初始化 L = 0, S = 0, Y, mu

  

7.      repeat（内层循环）

  

8.          % ---- L 更新：加权 SVT ----

9.          Z_L = PD + dt + (1/mu) * Y - S

10.         [U, Sigma, V] = svd(Z_L, 'econ')

11.         w_i = 1 若 sigma_i(L) < epsilon, 否则 w_i = 0

12.         L = U * diag( max(sigma_i(Z_L) - w_i/mu, 0) ) * V^T

  

13.         % ---- S 更新：L_{2,1} 行收缩 ----

14.         Z_S = PD + dt + (1/mu) * Y - L

15.         S_{i,:} = max(1 - lambda/(mu * ||Z_{S,i,:}||_2), 0) * Z_{S,i,:}

  

16.         % ---- dt 更新：QR 投影 ----

17.         dt_i = Q_i^T * (L + S - (1/mu)*Y - PD)_{:,i}

  

18.         % ---- 对偶变量更新 ----

19.         Y = Y + mu * (PD + dt - L - S)

20.         mu = rho * mu

  

21.     until 内层收敛（式 20）

  

22.     % ---- 更新几何参数 ----

23.     dt_i = R_i^{-1} * dt_i

24.     t_i(3) = t_i(3) + dt_i

  

25. until 外层收敛（式 22）

  

26. return L, S, t

```
